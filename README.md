# ShazamECG
SHAZAM ECG es un Sistema de Soporte a la Decisión Clínica IA. Utiliza redes neuronales 1D, aprendizaje contrastivo y búsquedas vectoriales FAISS en AWS para clasificar electrocardiogramas. Elimina la "caja negra" recuperando casos médicos históricos idénticos en milisegundos, brindando explicabilidad en Cuidados Intensivos.
